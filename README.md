\# SmartCare Hospital Management System (smartcare-hms)



A Spring Boot REST API for the SmartCare Hospital Management System, built as part of the CCS1303 Object-Oriented Programming coursework. Integrates with the existing MySQL database from the Database Systems module.



\## Tech Stack

\- Java 21

\- Spring Boot 3.x

\- Spring Data JPA / Hibernate

\- MySQL

\- Maven



\## Modules

\- Patient Management

\- Doctor Management

\- Department Management

\- Appointment Management

\- Admission \& Room Management

\- Treatment Management

\- Laboratory Management

\- Billing \& Payment Management



\## Getting Started



1\. Clone the repository

2\. Configure `src/main/resources/application.properties` with your MySQL credentials

3\. Run `mvn spring-boot:run` or run `SmartcareHmsApplication.java` from your IDE

4\. API runs on `http://localhost:8080`
